# Changelog

## 0.1.0

- Initial public release.
- Block elements:
  - Paragraphs
  - ATX headings
  - Horizontal rule
  - Blockquote
  - Lists (simple, single-line items)
  - Fenced code blocks
  - Tables (simple GFM-style)
- Inline elements:
  - Bold / italic
  - Links (with basic URL sanitization)
  - Inline code
  - Images
  - Strikethrough
